Project: Php Chart Helpers

Details:
The purpose of this project is to easily create Jquery/Javascript charts by simply using a php class.
I plan on expanding this project to include more Jquery/javascript libraries but right now I'm only 
using Charts.js found at http://www.chartjs.org/

This is more of a side project to my main project since my main project will utilize lots of charts.

I will update this readme with new jquery/javascript libraries I add
